## Definition

The most generic kind of creative work, including books, movies, photographs, software programs, etc. Non-expressive digital artifacts (binaries, model weights, dataset snapshots) are CreativeWork with refinement via Statements (artifactKind, mediaType, generatedBy, derivedFrom).

- **Boundaries:** Not a raw value or formal concept. Not a running agent instance (use SoftwareAgent). Not a resource where location is primary identifier (use NetworkResource).
- **Hex Code:** `30`
- **Standard Alignment:** [`schema:CreativeWork`](https://schema.org/CreativeWork) (Exact)
