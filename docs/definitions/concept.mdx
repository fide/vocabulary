## Definition

A formal concept, word, topic, category, or abstract idea defined in a taxonomy.

- **Boundaries:** Not a plain raw value.
- **Hex Code:** `31`
- **Standard Alignment:** [`skos:Concept`](https://www.w3.org/2004/02/skos/core#Concept) + [`schema:DefinedTerm`](https://schema.org/DefinedTerm) (Exact fit)

## Fide ID Rules

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
