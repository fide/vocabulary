## Definition

A tangible, inanimate physical object (e.g., hardware, vehicle, physical document, natural object). Physical only. Use Statements (madeBy, manufactured=true) if man-made vs natural matters.

- **Boundaries:** Not a location (where). Not a digital artifact (use CreativeWork).
- **Hex Code:** `43`
- **Standard Alignment:** [`dcmitype:PhysicalObject`](http://purl.org/dc/dcmitype/PhysicalObject) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when PhysicalObject is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`NetworkResource`](/vocabulary/definitions/network-resource).
