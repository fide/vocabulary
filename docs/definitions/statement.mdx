## Definition

A record of a subject-property-object assertion.

- **Boundaries:** Not a verified fact; verification is application-level. Not what it represents.
- **Hex Code:** `00`
- **Standard Alignment:** [`rdf:Statement`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Statement) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when Statement is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`Statement`](/vocabulary/definitions/statement).




### As Reference Type


<details>
<summary>[Reference](/fide-id/specification/syntax#reference-identifier) MUST use `subjectFideId|propertyFideId|objectFideId`, where the middle slot MUST reference a Property entity, in that order.</summary>

- Valid Reference:

```text
did:fide:0x10205fcbdc6d73bcfcd2c73eb4795c2f02f1d1c1|did:fide:0x3120580f7f68f7884e6be0603679f41b39d6d15b|did:fide:0xa0a0ba86d94f45ed925f5a31934b31c3d12c268a
```
- Invalid Reference:

```text
did:fide:0x10205fcbdc6d73bcfcd2c73eb4795c2f02f1d1c1|did:fide:0x3120580f7f68f7884e6be0603679f41b39d6d15b
```
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `subject|property|object`
</details>
