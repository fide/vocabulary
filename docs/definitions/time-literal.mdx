## Definition

A time of day (formatted according to ISO 8601, represented in UTC).

- **Boundaries:** Not a date or datetime.
- **Hex Code:** `b4`
- **Standard Alignment:** [`rdf:Literal`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Literal) (Broad) + [`xsd:time`](https://www.w3.org/2001/XMLSchema#time) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when TimeLiteral is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`TimeLiteral`](/vocabulary/definitions/time-literal) or [`NetworkResource`](/vocabulary/definitions/network-resource).




### As Reference Type

- [Reference](/fide-id/specification/syntax#reference-identifier) MUST use UTC time format with a trailing `Z` ([XML Schema 1.1 Part 2: Datatypes](https://www.w3.org/TR/xmlschema11-2/#time)).

<details>
<summary>Time references MUST use zero-padded hour, minute, and second values.</summary>

- Valid [Reference](/fide-id/specification/syntax#reference-identifier): `14:30:00Z`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `14:30:00`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `2:30 PM`
</details>
