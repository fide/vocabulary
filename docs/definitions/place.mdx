## Definition

A spatial context that can contain presence, co-location, and interaction. Includes physical locations and persistent virtual environments (e.g., a concert venue, a Discord server or channel, a Zoom room, a VRChat instance). Physicality/virtuality is asserted via Statements, not the base type.

- **Boundaries:** Not a happening or a physical artifact. Can an Actor have presence there? Is its identity a spatial address rather than a document location or account?
- **Hex Code:** `40`
- **Standard Alignment:** [`schema:Place`](https://schema.org/Place) (Broad) + [`schema:VirtualLocation`](https://schema.org/VirtualLocation) (Broad)
