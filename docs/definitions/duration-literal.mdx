## Definition

A calculated amount of elapsed time (ISO 8601 duration format).

- **Boundaries:** Not a datetime.
- **Hex Code:** `b5`
- **Standard Alignment:** [`rdf:Literal`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Literal) (Broad) + [`xsd:duration`](https://www.w3.org/2001/XMLSchema#duration) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when DurationLiteral is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`DurationLiteral`](/vocabulary/definitions/duration-literal) or [`NetworkResource`](/vocabulary/definitions/network-resource).




### As Reference Type

- [Reference](/fide-id/specification/syntax#reference-identifier) MUST use ISO 8601 duration format ([XML Schema 1.1 Part 2: Datatypes](https://www.w3.org/TR/xmlschema11-2/#duration)).

<details>
<summary>Duration references MUST begin with `P` and use valid duration components.</summary>

- Valid [Reference](/fide-id/specification/syntax#reference-identifier): `P3D`
- Valid [Reference](/fide-id/specification/syntax#reference-identifier): `PT1H30M`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `3 days`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `1h30m`
</details>
