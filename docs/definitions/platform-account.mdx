## Definition

An authority-based account principal (e.g., an email address or a specific GitHub, Google, or Mastodon user ID). The identity exists by the permission of a host system's database.

- **Boundaries:** Authenticated by a host authority's database. This represents the account principal itself, not the agent or agents that operate it.
- **Hex Code:** `21`
- **Standard Alignment:** [`schema:OnlineAccount`](https://schema.org/OnlineAccount) (Close)
