## Definition

A plain text sequence of characters.

- **Boundaries:** Not a Concept.
- **Hex Code:** `a0`
- **Standard Alignment:** [`rdf:Literal`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Literal) (Broad) + [`xsd:string`](https://www.w3.org/2001/XMLSchema#string) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when TextLiteral is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`TextLiteral`](/vocabulary/definitions/text-literal) or [`NetworkResource`](/vocabulary/definitions/network-resource).




### As Reference Type

- [Reference](/fide-id/specification/syntax#reference-identifier) MUST use the exact text value.
