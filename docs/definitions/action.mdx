## Definition

A discrete assertion or commitment attributable to an Agent. Often produces or records a change (e.g., a transaction, API call, signature). Can be linked to Events via Statements (causes, records, resultsIn).

- **Boundaries:** Not a broad temporal happening (Event); must be driven by an Agent.
- **Hex Code:** `42`
- **Standard Alignment:** [`schema:Action`](https://schema.org/Action) (Close) + [`prov:Activity`](https://www.w3.org/ns/prov#Activity) (Close)
