## Definition

A math-based account principal (e.g., an EVM address, Bitcoin wallet, or did:key). The identity exists mathematically and is authenticated via mathematical signature keys.

- **Boundaries:** Authenticated via mathematical signature keys, independent of a host authority.
- **Hex Code:** `22`
- **Standard Alignment:** [`sec:PublicKey`](https://w3id.org/security#PublicKey) (Close) + [`sec:publicKeyMultibase`](https://w3id.org/security#publicKeyMultibase) (Close)
