## Definition

A decimal or floating-point number.

- **Boundaries:** Not an integer.
- **Hex Code:** `a2`
- **Standard Alignment:** [`rdf:Literal`](https://www.w3.org/1999/02/22-rdf-syntax-ns#Literal) (Broad) + [`xsd:decimal`](https://www.w3.org/2001/XMLSchema#decimal) (Exact)


---

## Fide ID Rules

These entity-specific constraints apply when DecimalLiteral is used in [Fide ID](/fide-id).

### As Entity Type

- MUST use [Reference Type](/fide-id/specification/syntax#reference-type) [`DecimalLiteral`](/vocabulary/definitions/decimal-literal) or [`NetworkResource`](/vocabulary/definitions/network-resource).




### As Reference Type

- [Reference](/fide-id/specification/syntax#reference-identifier) MUST use a base-10 decimal string ([XML Schema 1.1 Part 2: Datatypes](https://www.w3.org/TR/xmlschema11-2/#decimal)).

<details>
<summary>Decimal references MUST NOT use scientific notation or thousands separators.</summary>

- Valid [Reference](/fide-id/specification/syntax#reference-identifier): `4.2`
- Valid [Reference](/fide-id/specification/syntax#reference-identifier): `-0.5`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `1e3`
- Invalid [Reference](/fide-id/specification/syntax#reference-identifier): `1,000.50`
</details>
